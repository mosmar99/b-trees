searchElement(TREE, VAL) 

STEP 1: IF TREE->DATA = VAL OR TREE = NULL, then 
            Return TREE
        ELSE
            IF VAL < TREE->DATA
                Return search(TREE->LEFT, VAL)
            ELSE 
                Return search(TREE->RIGHT, VAL)
            [ENF IF]
        [END IF]

STEP 2 END
